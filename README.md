## рҹ“– Overview

This repository presents an end-to-end deep learning framework for solving an important **optical inverse problem**:

> **Estimating the wavelength of incident light directly from its Fraunhofer diffraction intensity pattern using a Convolutional Neural Network (CNN) implemented in PyTorch.**

Traditional optical phase retrieval and inverse reconstruction methods often rely on computationally expensive iterative algorithms such as:

* GerchbergвҖ“Saxton Algorithm
* Fienup Phase Retrieval
* Hybrid Input-Output (HIO)
* Error Reduction Methods

While highly accurate, these approaches may not be suitable for real-time optical sensing applications.

This project demonstrates how a carefully designed CNN can learn the nonlinear mapping between diffraction images and physical parameters, enabling extremely fast inference once trained.

---

# рҹҺҜ Project Motivation

The primary objective of this work is to investigate whether a neural network can learn physical laws implicitly and infer optical parameters directly from diffraction measurements without explicitly solving the inverse wave propagation problem.

# рҹ“җ Forward Physical Model

Under the **Fraunhofer diffraction approximation**, the intensity distribution produced by a circular aperture follows the Airy pattern:

$$
I(\theta)=I_0
\left(
\frac{2J_1(kR\sin\theta)}
{kR\sin\theta}
\right)^2
$$

where

| Symbol    | Meaning                     |
| --------- | --------------------------- |
| $I_0$     | Peak intensity              |
| $J_1$     | First-order Bessel function |
| $R$       | Aperture radius             |
| $k$       | Optical wave number         |
| $\lambda$ | Incident wavelength         |
| $\theta$  | Observation angle           |

and

$$
k=\frac{2\pi}{\lambda}
$$
